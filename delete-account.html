Como pedir a eliminação da conta

Para pedir a eliminação da tua conta AtuAgenda e de todos os dados associados, envia um email com o seguinte:

  • Assunto: Eliminação de Conta AtuAgenda
  • Conteúdo: o email associado à tua conta (Google ou email/password)

O processo é concluído no prazo de 30 dias após a confirmação do pedido.

Dados que são eliminados

Quando a conta é eliminada, os seguintes dados são permanentemente removidos da base de dados Firebase:

  • Todos os compromissos e agendamentos
  • Histórico de clientes e notas de clientes
  • Notas pessoais e marcadores de dias
  • Configurações de serviços e horários de trabalho
  • Dados do plano (gratuito ou premium)
  • Conta de autenticação Firebase
⚠️ A eliminação da conta é irreversível. Recomendamos que exportes um backup dos teus dados antes de pedir a eliminação: Menu → Backup → Exportar Backup.

Nenhum dado pessoal é mantido após a eliminação. Todos os dados são removidos no prazo de 30 dias.

🔧

Alternativa: eliminar só os dados

Se pretenderes eliminar apenas os teus dados sem eliminar a conta, podes fazê-lo diretamente na app:

  • Abre o menu de utilizador (avatar no canto superior direito)
  • Vai a Dados → Limpar período
  • Seleciona o intervalo de datas e confirma a eliminação

Esta opção elimina os compromissos do período selecionado mas mantém a conta ativa.
