Eliminação de Conta e Dados

🗑️ Como pedir a eliminação da conta

Para pedir a eliminação da tua conta AtuAgenda e de todos os dados associados, segue estes passos:

  1. Envia um email para o endereço indicado abaixo com o assunto "Eliminação de Conta AtuAgenda"
  2. Indica o email associado à tua conta (Google ou email/password)
  3. Aguarda a confirmação — o processo é concluído no prazo de 30 dias

Envia o pedido para:

[email protected]

📦 Dados que são eliminados

Quando a conta é eliminada, os seguintes dados são permanentemente removidos da base de dados Firebase:

  • Todos os compromissos e agendamentos
  • Histórico de clientes e notas
  • Notas pessoais e marcadores de dias
  • Configurações de serviços e trabalho
  • Dados do plano (gratuito ou premium)
  • Conta de autenticação Firebase
Compromissos Clientes Notas Configurações Conta

📋 Dados que podem ser mantidos

Nenhum dado pessoal é mantido após a eliminação da conta. Os dados são eliminados na totalidade no prazo de 30 dias após a confirmação do pedido.

⚠️ A eliminação da conta é irreversível. Recomendamos que exportes um backup dos teus dados antes de pedir a eliminação (Menu → Backup → Exportar Backup).

❓ Alternativa: eliminar só os dados

Se pretenderes eliminar apenas os teus dados sem eliminar a conta, podes fazê-lo diretamente na app:

  1. Abre o menu de utilizador (avatar no canto superior direito)
  2. Vai a Dados → Limpar período para eliminar compromissos de um período
  3. Para eliminar tudo, seleciona o intervalo completo de datas
