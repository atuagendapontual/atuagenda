Quem somos

A AtuAgenda é uma aplicação Android de gestão de agendamentos desenvolvida para profissionais independentes portugueses — cabeleireiros, esteticistas, barbeiros, personal trainers e outros prestadores de serviços baseados em marcações.

O responsável pelo tratamento de dados é o desenvolvedor da aplicação, contactável através do email [email protected].

📋

Dados recolhidos

A AtuAgenda recolhe apenas os dados estritamente necessários para o funcionamento da aplicação:

  • Dados da conta Google: email e nome associados à conta usada para autenticação. Não acedemos a contactos, emails, calendários ou qualquer outro dado da conta Google.
  • Dados de agenda: nomes de clientes, números de telefone (opcionais), datas, horas, serviços e valores que tu próprio introduzes na app.
  • Notas e preferências: notas internas sobre clientes e preferências de serviço que decides registar.
  • Preferências da app: tema de cor escolhido e ativação do modo escuro.
ℹ️ Não recolhemos dados de localização, não acedemos à câmara, microfone, contactos do telemóvel, nem a qualquer sensor do dispositivo.
⚙️

Como usamos os dados

Os dados recolhidos são usados exclusivamente para:

  • Permitir o funcionamento da app — mostrar a tua agenda, histórico de clientes e balanço financeiro.
  • Sincronizar os dados entre dispositivos através da cloud Firebase.
  • Enviar notificações de lembretes de compromissos, caso atives esta funcionalidade.
  • Preservar as tuas preferências de interface (tema, modo escuro).

Não usamos os teus dados para fins publicitários, análise de comportamento, machine learning, nem para qualquer outro propósito comercial.

⚡

Planos gratuito e premium

A AtuAgenda está disponível em duas versões:

  • Plano Gratuito: permite criar até 20 compromissos no total e guardar até 10 clientes. Este limite é acumulado e não repõe — mesmo que apagues compromissos, o contador não desce. Este mecanismo é guardado de forma segura na cloud e também localmente, funcionando mesmo sem internet.
  • Plano Premium (7,99€ único): acesso ilimitado a todas as funcionalidades, incluindo compromissos ilimitados, backup automático, histórico completo, balanço anual e lembretes.
ℹ️ O contador de compromissos criados é guardado no Firebase e em cache local — desinstalar e reinstalar a app não repõe este contador.
☁️

Onde guardamos os dados

Os dados são guardados em dois locais:

  • Firebase Firestore (Google Cloud): os dados da tua agenda, clientes e notas são guardados na cloud Firebase, associados ao teu UID de utilizador. Os servidores Firebase estão localizados na União Europeia ou nos EUA, conforme a configuração do projeto.
  • Dispositivo local (cache): após o primeiro login, uma cópia dos dados é guardada no armazenamento local do teu telemóvel. Isto permite que a app funcione mesmo sem ligação à internet — os dados são sincronizados automaticamente quando a ligação for restabelecida. A sessão de login também é guardada localmente, permitindo abrir a app sem internet sem necessidade de novo login.
🔐 Cada utilizador só consegue aceder aos seus próprios dados. As regras de segurança do Firebase garantem que nenhum utilizador pode ler ou modificar dados de outro utilizador.
🤝

Partilha de dados

Não vendemos, não alugamos e não partilhamos os teus dados com terceiros.

Os únicos serviços de terceiros que têm acesso técnico aos dados são:

  • Google Firebase: plataforma de base de dados e autenticação. Política de privacidade disponível em firebase.google.com/support/privacy.
  • Google Sign-In: usado para autenticação. A Google apenas confirma a identidade — não partilha dados da conta com a app além do email e nome.

Estes serviços são utilizados exclusivamente para o funcionamento técnico da app, não para fins de marketing ou análise.

⚖️

Os teus direitos (RGPD)

Ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD), tens os seguintes direitos:

  • Acesso: podes consultar todos os dados que guardaste na app a qualquer momento, diretamente na interface.
  • Exportação: podes exportar todos os teus dados em formato JSON através da função Backup nas definições da app.
  • Retificação: podes editar ou corrigir qualquer dado diretamente na app.
  • Eliminação: podes eliminar qualquer compromisso, nota ou cliente diretamente na app. Para eliminar a conta completa e todos os dados associados, contacta-nos por email.
  • Portabilidade: os dados exportados via Backup estão em formato JSON legível, podendo ser importados para outros sistemas.
  • Oposição: podes deixar de usar a app e solicitar a eliminação de todos os dados a qualquer momento.
🗓️

Retenção de dados

Os dados são mantidos enquanto tiveres uma conta ativa na app. Se solicitares a eliminação da conta, todos os dados associados ao teu UID serão eliminados permanentemente da base de dados Firebase no prazo de 30 dias.

A cache local no teu dispositivo pode ser eliminada a qualquer momento através das definições do Android em Definições → Apps → AtuAgenda → Armazenamento → Limpar dados. Note que limpar os dados da app elimina a sessão guardada — será necessário fazer login novamente na próxima utilização.

🛡️

Segurança

Implementámos as seguintes medidas de segurança para proteger os teus dados:

  • Autenticação obrigatória — nenhum dado é acessível sem login.
  • Regras de segurança Firebase que isolam cada utilizador — nenhum utilizador acede a dados de outro.
  • Comunicação encriptada HTTPS entre a app e os servidores Firebase.
  • Sem armazenamento de passwords — a autenticação é delegada inteiramente à Google.
👶

Menores de idade

A AtuAgenda é uma aplicação destinada a uso profissional e não é direcionada a menores de 16 anos. Não recolhemos conscientemente dados de menores. Se tomares conhecimento de que um menor nos forneceu dados pessoais, contacta-nos para que possamos proceder à sua eliminação imediata.

📝

Alterações a esta política

Podemos atualizar esta política de privacidade ocasionalmente para refletir alterações na app ou na legislação aplicável. Quando o fizermos, atualizaremos a data de "última atualização" no topo desta página.

Para alterações significativas, notificaremos os utilizadores ativos através de um aviso dentro da app. Recomendamos que consultes esta página periodicamente.

✉️

Contacto

Para qualquer questão relacionada com privacidade, pedidos de eliminação de dados ou exercício dos teus direitos ao abrigo do RGPD:

AtuAgenda — Privacidade

Respondemos a todos os pedidos no prazo máximo de 72 horas.
