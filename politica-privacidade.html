Política de Privacidade

Versão 1.0 Última atualização: Janeiro 2026

📋 Resumo simples: A AtuAgendaPontual recolhe apenas os dados estritamente necessários para o funcionamento da app. Nunca vendemos, partilhamos ou usamos os teus dados para publicidade.

1. Quem somos

A AtuAgendaPontual é uma aplicação Android desenvolvida de forma independente para gestão de agendamentos de pequenos negócios. Para questões de privacidade, podes contactar-nos através do email indicado no final deste documento.

2. Dados que recolhemos

A AtuAgendaPontual recolhe apenas os dados que introduzes diretamente na aplicação:

  • Dados de conta: endereço de email e palavra-passe (para autenticação via Firebase Auth)
  • Dados de negócio: nomes de clientes, números de telefone, datas e horas de agendamentos, valores, notas e serviços — introduzidos por ti
  • Dados técnicos: identificador único de dispositivo para funcionamento do sync na cloud

Não recolhemos dados de localização, contactos do telemóvel, câmara, microfone ou qualquer outro dado sem o teu conhecimento.

3. Como usamos os teus dados

Os teus dados são usados exclusivamente para:

  • Guardar e sincronizar os teus agendamentos entre dispositivos
  • Permitir o acesso à tua conta em múltiplos dispositivos
  • Apresentar os dados na interface da aplicação

Os teus dados nunca são usados para publicidade, análise de comportamento, partilhados com terceiros para fins comerciais, ou vendidos.

4. Onde ficam guardados os dados

Os teus dados são armazenados de forma segura no Google Firebase (Firestore Database e Firebase Authentication), em servidores localizados na Europa (região europe-west1). O Firebase é um serviço da Google LLC e está em conformidade com o RGPD.

Os dados são igualmente guardados localmente no teu dispositivo (localStorage) para funcionamento offline.

5. Segurança

Implementamos as seguintes medidas de segurança:

  • Autenticação obrigatória com email e palavra-passe
  • Cada utilizador só tem acesso aos seus próprios dados (regras de segurança do Firestore)
  • Comunicação cifrada via HTTPS/TLS
  • Palavra-passe armazenada de forma cifrada (nunca em texto simples)

6. Partilha de dados com terceiros

Utilizamos os seguintes serviços de terceiros para o funcionamento da app:

  • Google Firebase — autenticação e base de dados na cloud. Política de privacidade da Firebase
  • Google Play Billing — processamento do pagamento único da app. Política de privacidade do Google Pay

Não partilhamos dados com quaisquer outros terceiros.

7. Os teus direitos (RGPD)

Ao abrigo do Regulamento Geral de Proteção de Dados (RGPD), tens direito a:

  • Acesso: solicitar uma cópia de todos os dados que guardamos sobre ti
  • Retificação: corrigir dados incorretos (podes fazê-lo diretamente na app)
  • Eliminação: solicitar a eliminação da tua conta e todos os dados associados
  • Portabilidade: exportar os teus dados em formato JSON (função disponível na app)
  • Oposição: opor-te ao tratamento dos teus dados em qualquer momento

Para exercer qualquer um destes direitos, contacta-nos pelo email abaixo.

8. Retenção de dados

Os teus dados são guardados enquanto mantiveres uma conta ativa na AtuAgendaPontual. Podes eliminar a tua conta e todos os dados associados a qualquer momento, contactando-nos por email. Após eliminação, os dados são removidos permanentemente em até 30 dias.

9. Crianças

A AtuAgendaPontual não é destinada a menores de 16 anos e não recolhe conscientemente dados de crianças. Se tomares conhecimento de que uma criança nos forneceu dados pessoais, contacta-nos imediatamente.

10. Alterações a esta política

Podemos atualizar esta Política de Privacidade ocasionalmente. Quando o fizermos, atualizaremos a data no topo deste documento e, se as alterações forem significativas, notificaremos os utilizadores por email.
